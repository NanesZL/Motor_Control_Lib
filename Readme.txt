以下使用的Matlab为2022b版本

BLDC_LADRC_Motor_Speed_Control
BLDC电机线性ADRC速度电流双闭环，速度环使用LADRC，电流环使用PI控制母线电流，母线电流来自于供电母线，包含正反相换向

BLDC_Back_EMF
观察BLDC电机的反电动势

BLDC_Control_Buck_Converter
BLDC电机单速度环，PWM控制母线电压来控制速度

BLDC_Hysteresis_Current_Loop
BLDC电机电路滞环控制，控制母线电流，电流反馈来自于三相电流，分别对三相电流进行反馈闭环，六步换向控制电机

BLDC_Hysteresis_Speed_Current_Loop
BLDC电机电路滞环控制，速度电流环

BLDC_PWM_Control
BLDC电机速度单环PWM控制，采用六步换向控制

BLDC_SixStep_Speed_Control
BLDC电机六步换向控制，只有速度环，包含六步换向逻辑表等

BLDC_SixStep_Speed_Current_Loop_Control
BLDC电机六步换向速度电流双环控制（包含但电流环），只有正向控制，电流闭环反馈电流用的是三相电流绝对值求和的一半。

BLDC_Speed_Torque_Current_ClosedLoop
BLDC电机速度力矩电流三闭环（包含力矩、电流闭环），六步换向，正反转换向，最外环为速度环、中间环为力矩环、最内环为电流环，速度环ADRC（Simulink自带），力矩电流环均为PI控制

BLDC_Speed_Torque_Current_ClosedLoop_LowSample
BLDC电机速度力矩电流三闭环低采样率和控制频率，为20KHz

BLDC_Position_Speed_Current_loop
BLDC电机位置速度电流三闭环控制，六步换向


DTP_PMSM
双三相永磁同步电机模型

DTP_PMSM_Dual_DQ_Control
基于双 d-q 坐标变换的DTP-PMSM矢量控制

DTP_PMSM_VSD_Four_Vector_Control
基于 VSD 坐标变换的DTP-PMSM 矢量控制



Luenberger_Observer
龙伯格观测器仿真

PMSM_3PhaseCurrent
PMSM电机模拟三相正弦电流，只有三个相差120°正弦信号

PMSM_DTC_Traditional_Control
PMSM电机传统直接转矩控制，速度闭环


PMSM_FOC_Algorithm
FOC算法，包括SVPWM和PMSM电机模型

PMSM_FOC_CyclicSpeed_Closedloop
PMSM电机FOC周期变转速闭环控制，来源于硕士毕业论文

PMSM_FOC_Openloop
PMSM电机FOC开环控制

PMSM_FOC_Speed_Closedloop（old）
PMSM电机FOC速度环控制，老版本，不建议使用

PMSM_FOC_Speed_PID
PMSM电机FOC速度环PID控制，但是PID输出Ud、Uq部分加入了电流和磁链

PMSM_FOC_Speed_PID_parametertune
PMSM电机FOC速度环PID控制，最建议使用的模板

PMSM_FOC_Speed_PID_Udq
PMSM电机FOC速度环PID控制，PID输出就是Ud、Uq

PMSM_FOC_Speed_PR
PMSM电机静止坐标系下比例谐振控制

PMSM_MPCC
PMSM电机模型预测电流控制，速度闭环

PMSM_Sensorless_ASMO_PLL
PMSM电机使用自适应滑膜+PLL（锁相环）位置估计，仿真效果还行，其他转速尤其是低速可能还要再调参数

PMSM_Sensorless_MARS_idq
PMSM电机使用模型参考自适应，直接对idq进行观测，来估算位置，角度估算都有个延迟，还没解决

PMSM_Sensorless_MARS_idq_prime
PMSM电机使用模型参考自适应，先进行了一步变换idq_prime，再对idq_prime进行观测，来估算位置，角度估算都有个延迟，还没解决

PMSM_Sensorless_SOGI_PLL
PMSM电机使用二阶广义积分器+PLL（锁相环）位置估计，启动时初始位无法估计，因为无初始位置转子信息，且速度太低，反电势噪声干扰太大

PMSM_Sensorless_Traditional_SMO_arctan
PMSM电机无感传统滑膜观测器观测反电动势，反正切（锁相环）位置估计，有问题（不建议用，建议用PLL）

PMSM_Sensorless_Traditional_SMO_PLL
PMSM电机无感传统滑膜观测器观测反电动势，PLL（锁相环）位置估计，存在位置估算上的相位延迟，闭环效果有待优化

PMSM_SMC_Speed_Closedloop
PMSM电机速度环滑模控制
